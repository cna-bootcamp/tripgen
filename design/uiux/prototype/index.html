주요 기능

🎨 디자인 시스템

  • 일관된 컬러 팔레트와 타이포그래피
  • 재사용 가능한 UI 컴포넌트
  • 반응형 웹 디자인 (320px ~ 1280px+)
  • 접근성 고려 (WCAG 2.1 AA)

⚡ 인터랙션

  • 실시간 폼 유효성 검사
  • 부드러운 애니메이션과 전환 효과
  • 토스트 알림 및 모달 다이얼로그
  • 키보드 네비게이션 지원

🔧 기술 스택

  • 순수 HTML5, CSS3, JavaScript
  • CSS Variables 기반 테마 시스템
  • ES6+ 모던 JavaScript
  • 공통 컴포넌트 라이브러리

📱 사용자 경험

  • 직관적인 사용자 플로우
  • 모바일 퍼스트 디자인
  • 로딩 상태 및 스켈레톤 UI
  • 오프라인 대응 및 에러 처리

사용 가이드

🚀 시작하기

대시보드에서 "새 여행 계획하기" 버튼을 클릭하여 여행 일정 생성을 시작하세요. 단계별 가이드가 제공됩니다.

📝 프로파일 설정

멤버 정보, 여행 일정, 이동수단을 차례로 설정하면 AI가 개인화된 여행 일정을 생성합니다.

🤖 AI 일정 생성

Claude API를 활용한 맞춤형 일정 생성. 실시간 진행률을 확인하고 원하지 않으면 언제든 취소할 수 있습니다.

🗺️ 지도 및 리뷰

생성된 일정을 지도에서 확인하고, 각 장소의 리뷰와 평점을 통해 여행 계획을 완성하세요.
